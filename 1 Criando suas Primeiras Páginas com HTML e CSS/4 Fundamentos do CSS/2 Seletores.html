Seletores CSS

Bem-vindo!

No CSS nós possuímos os seletores por ID:

  • Seletor por Tipo: Busca elementos por uma tag HTML;
  • Seletor por ID (#): Busca elementos atrás do atribuito "id";
  • Seletor por Classe (.): Busca elementos atrás do atribuito "class";
  • Seletor Universal (*): Seleciona todos os elementos HTML;
  • Seletor de Atributo ([atrib]): Seleciona todos os elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico.
